肺炎

  • ❆儀器

    • 胸部Ｘ光 : 藉由Ｘ光對我們人體各種不同組織的穿透性不一，將胸腔內原來的三度空間結構投影顯像在一張平面的Ｘ光片或偵測器上，以檢測胸部是否有病變的檢查。
    • 血液學檢查（ 血液培養、血清學檢查、分子生物學檢驗、生物指標） :
      檢測項目 : 尿素氮 > 20mg/dL、血壓是否太低（收縮壓 < 90 mmHg或舒張壓 ≤ 60 mmHg）、aO2 / FiO2 比值 < 250...等基本生理指標；
      呼吸道融合病毒、腺病毒、肺炎鏈球菌、隱球菌、肺囊蟲、革蘭氏陽性菌...等病菌、病毒；
      C- 反應蛋白、前降鈣素、前腎上腺髓質素...等生物標記相關。
    • 尿液抗原檢驗 : 檢查快速（約 15 分鐘）、簡單、特異性高，且在病人接受抗生素治療後仍有機會診斷病因。
    • 支氣管鏡 : 為一種侵襲性檢查，先為病人咽喉局部麻醉，再經由口腔或鼻腔放入一根直徑約0.6公分的內視鏡，進行檢查。
