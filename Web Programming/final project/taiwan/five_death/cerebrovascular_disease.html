腦血管疾病

  • ❆介紹

    • 腦梗塞 : 俗稱腦中風，是為一種急症，主要是因血管或身體其他部位血液內的雜質或血塊，被血流沖落形成栓子，
            導致無法供應腦部氧氣，腦組織壞死和功能失調，常見有腦血栓症及腦栓塞症兩種。
    • 暫時性腦缺血 : 腦血管病變引起的短暫性、局限性腦功能缺失或視網膜功能障礙(24hr內腦功能異常完全恢復)。
    • 粥狀動脈硬化 : 因油脂累積、內膜增厚而形成粥狀斑塊，使得血管硬化、狹窄、甚至阻塞於腦部的病理現象。
    • 腦部血管畸形 : 因先天腦血管異常，其中最多的是先天性腦部動靜脈畸形，動脈沒有經過微血管減壓就直接進入靜脈，造成靜脈漲大，久了就造成腦出血。
    • 頭頸部動脈剝離 : 頸動脈剝離指的是內頸動脈的內膜因為某種原因產生了一個裂縫，於是血液就鑽入了血管壁，而把血管壁剝離開了，而這些鑽入血管壁的血液就把血管的管腔擠扁。
    • 顱內出血 : 指頭骨內有血液蓄積，血塊可能累積在大腦組織內或頭骨底下，壓迫腦部，並可能導致生命危險。

      顱內出血分成2種：
      硬腦膜上出血：指血腫位於顱骨與硬腦膜(大腦外最外層的腦膜)之間。
      硬腦膜下出血：指血腫位於硬腦膜與腦組織之間。

  • ❆治療

    • 藥物治療 : 通常以「抗血小板藥物」及「抗凝血藥物」(通常統稱為「抗血栓藥物」)來治療及預防再次中風。
    • 腦血管栓塞手術 : 病人於麻醉狀態下，經穿刺股動脈將導管放置於病變處，先以傳統血管攝影確定腦血管病變位置，再將微細導管置入病變或病灶處，
                  並選擇適當之栓塞物（如：高尼米 可分離線圈），在X光透視下將栓塞物置入病變或病灶，阻斷這些不正常之腦血管病 灶，以達到治療目的。
